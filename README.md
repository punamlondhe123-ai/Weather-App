# Weather Map API Web Application

## Description
This is a simple web-based Weather Application that shows current weather conditions and a 5-day forecast for any city using the OpenWeatherMap API.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- OpenWeatherMap API

## How to Run the Project
1. Download and extract the ZIP file
2. Open the folder
3. Double-click `myweather.html` or open it in any modern web browser
4. Enter a city name and click **Get Weather**

## Notes
- Internet connection is required
- You can replace the API key in `weather.js` with your own OpenWeatherMap API key if needed
- This project is suitable for mini or final-year projects
